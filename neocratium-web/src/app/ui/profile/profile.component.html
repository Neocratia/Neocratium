Bienvenido a Neocratium
Una herramienta para que los ciudadanos decidan que leyes, proyectos y acciones los gobiernos deben trabajar usando la democracia liquida.

Una democracia participativa donde cada ciudadano puede votar cada que haya una decisión que tomar, y puede delegar su voto en un/una congresista diferente para cada decisión. El voto de cada congresista tendrá el peso del número de ciudadanos que hayan delegado en él esa decisión.
